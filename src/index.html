Doodle uses a Convolutional Neural Network trained on a subset of the Quick Draw Google AI Dataset. Model is loaded in browser using TensorFlowJS. Accuracy is sacrificed for limited browser memory. Written with Vanilla JS, jQuery, and Webpack.
